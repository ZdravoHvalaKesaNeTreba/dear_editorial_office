# Адаптация дизайна под Яндекс Старт / Яндекс Реклама

## Обзор изменений

Дизайн проекта "Дорогая Редакция" был адаптирован под фирменный стиль Яндекс Старт и Яндекс Реклама, включая:

### 1. Типографика

**Шрифты:**
- **YS Text** (400, 500) - для основного текста
- **YS Display** (700, 800) - для заголовков

**Размеры:**
- Title1: 54px / 60px (h1)
- Title2: 34px / 40px (h2)
- Body: 18px / 28px (p, основной текст)

### 2. Цветовая палитра

**Основные цвета:**
- `#FC0` - Яндекс желтый (акцент)
- `#191E28` - Яндекс черный (основной текст)
- `#21272E` - Яндекс темный (фон)
- `#5F6466` - Яндекс серый (второстепенный текст)
- `#DDC8FF` - Яндекс фиолетовый светлый (описания на темном фоне)

### 3. Компоненты

#### Кнопки
- Border radius: 16-24px
- Padding: 12-16px 24-32px
- Цвет: `#FC0` с hover эффектом `#FFD633`
- Font weight: 500-600

#### Карточки
- Border radius: 32px
- Padding: 24-32px
- Background: белый с тенью
- Hover: transform translateY(-4px) + shadow

#### Inputs/Textarea
- Border radius: 24px
- Background: светло-серый (#F5F5F5)
- Border: 1-2px solid с динамическим цветом

### 4. Структура секций

#### Секция "Для кого"
```css
.for-whom-section {
  max-width: 1440px;
  padding: 60px 78px 90px;
  background: #191E28;
}

.for-whom-card {
  width: calc(50% - 6px);
  height: 240px;
  border-radius: 32px;
  gap: 12px;
}
```

### 5. Темная тема

Используется градиент от `#191E28` до `#21272E` для создания глубины.

### 6. Анимации

- Transition: 0.2-0.3s ease
- Hover эффекты на кнопках и карточках
- Fade in animations для контента

## Применение в компонентах

### Header
- Белый текст на темном фоне
- Описания в цвете `#DDC8FF`
- Увеличенные размеры заголовков

### ServiceSelector
- Полупрозрачный фон (white/10)
- Border radius: 20px
- Hover эффекты с изменением прозрачности

### TextEditor
- Border radius: 24px
- Желтая кнопка "Проверить"
- Динамические borders (зеленый/красный)

### ResultsPanel
- Увеличенные иконки (64px)
- Rounded элементы (20px)
- Цветные индикаторы (красный/зеленый)

### Footer
- Желтые акценты на ссылках
- Hover эффекты с transition
- Адаптивная типографика

### ForWhom (новый компонент)
- Сетка 2x2 с gap 12px
- Белые карточки на темном фоне
- Title2 и Body типографика

- Tablet: 768px+
- Mobile: 320px+
